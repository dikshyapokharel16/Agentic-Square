# Agentic Square — Chat Script
# ---------------------------------------------------------------------
# Edit this file freely: reword messages, add new ones, remove ones,
# reorder them. When you're done, ask Claude to "sync the chat script"
# and it'll read this file and update messages.json to match.
#
# Each message is one block, separated by a blank line and starting
# with "---". The first line of a block is TYPE, sender (if any), and
# time. Everything after is the message content.
#
# Types:
#   DATE hh:mm            -> a date divider, e.g. "Monday"
#   SYSTEM hh:mm          -> a small centered system note
#   EVENT hh:mm           -> like SYSTEM but highlighted (accent color)
#   MSG Sender hh:mm      -> a normal chat message
#   MSG Sender hh:mm (me) -> a message sent by "you" (right-aligned)
#   IMAGE Sender hh:mm    -> shares a picture — ALSO ADVANCES which stage/
#                            3D model the AR panel shows, in the order
#                            these IMAGE blocks appear top to bottom.
#   FILE Sender hh:mm     -> shares a file; first line = filename, second
#                            (optional) line = subtitle
#   POLL Sender hh:mm     -> a poll; question on the first content line,
#                            then one "- option text" line per option
#
# Reactions/vote counts/colors are handled automatically — just focus on
# the words. If you rename a sender, they'll get a new consistent color.
# ---------------------------------------------------------------------
